## Usage Examples

### Scenario 1: Budget-Conscious Bug Bounty Hunter

**Recommendation:** Grok only

```bash
python subgrab.py target.com --grok-key xai-xxxxx
```

**Benefits:**
- Completely FREE
- High-quality AI assistance
- No ongoing costs

---

### Scenario 2: Professional Penetration Tester

**Recommendation:** Grok + OpenRouter (Dual AI)

```bash
python subgrab.py target.com \
  --grok-key xai-xxxxx \
  --openrouter-key sk-or-xxxxx \
  --openrouter-model anthropic/claude-3.5-sonnet
```

**Benefits:**
- Maximum coverage
- Two AI models validate findings
- Best possible results
- Worth the OpenRouter cost for critical engagements

---

### Scenario 3: Quick Reconnaissance

**Recommendation:** Built-in methods only

```bash
python subgrab.py target.com --fast --threads 100
```

**Benefits:**
- Fastest execution
- No API setup needed
- Good for initial recon

## When to Use What?

### Use Grok Alone When:
- ✅ You want FREE AI assistance
- ✅ Budget is a primary concern
- ✅ You're doing regular scans
- ✅ Learning subdomain enumeration
- ✅ Bug bounty hunting

### Use OpenRouter Alone When:
- ✅ You already have OpenRouter credits
- ✅ You prefer a specific model (Claude, GPT-4)
- ✅ You don't want to manage multiple API keys

### Use Both (Dual AI) When:
- ✅ Running critical assessments
- ✅ Maximum coverage is required
- ✅ Client deliverables demand thoroughness
- ✅ Comparing AI model outputs
- ✅ Validating findings across models

### Use Neither When:
- ✅ Quick reconnaissance needed
- ✅ No API keys available
- ✅ Testing tool functionality
- ✅ Simple/small domains

## Getting Started

### 1. Quick Start with Grok (Recommended)

```bash
# Sign up at console.x.ai (FREE)
# Get your API key
python subgrab.py target.com --grok-key xai-xxxxx
```

### 2. Add OpenRouter for Maximum Power

```bash
# Sign up at openrouter.ai ($5-10)
# Get your API key
python subgrab.py target.com \
  --grok-key xai-xxxxx \
  --openrouter-key sk-or-xxxxx
```

### 3. Configure Once, Use Forever

Create `api_keys.json`:
```json
{
  "grok": "xai-xxxxx",
  "openrouter": "sk-or-xxxxx"
}
```

Then just run:
```bash
python subgrab.py target.com
```
